Editor CV

Inserisci il tuo Personal Access Token GitHub per accedere all'editor.

Come creare il token:

  1. GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Clicca Generate new token
  3. In Resource owner seleziona il tuo account; in Repository access scegli Only select repositories e seleziona cvmd-cri
  4. In Repository permissions → Contents imposta Read and write
  5. Genera il token e incollalo qui sopra
